# 🚀 Developing with AntiGravity

Sentinel is designed to be easily extended and maintained. For the best development experience, we recommend using **AntiGravity**, a powerful agentic AI coding assistant.

### How to use AntiGravity with Sentinel:

1.  **Open AntiGravity**: Launch the AntiGravity application on your machine.
2.  **Import the Repository**: Select "Import Project" or open the `Sentinel` folder directly.
3.  **Start Coding**: Once indexed, you can ask AntiGravity to:
    *   "Implement a new plugin that tracks X..."
    *   "Debug the issue with Y..."
    *   "Refactor the Z service for better performance..."
4.  **Pair Program**: AntiGravity will work alongside you, providing real-time code generation, debugging, and architectural advice tailored to the Sentinel codebase.
